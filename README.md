# QuickSwitch

Dalamud plugin for quickly switching between your FFXIV characters.

## Installation

Add this custom plugin repository URL in Dalamud settings:

```
https://raw.githubusercontent.com/hocng015/QuickSwitch-Release/master/QuickSwitch.json
```

## Features

- Auto-detect characters on login
- One-click character switching
- Quick logout to character select
- Safety guards (cooldowns, combat/duty/cutscene blocking)
- Discord bot integration for remote control
- Admin controls (whitelist/blacklist, force switch, audit logging)
